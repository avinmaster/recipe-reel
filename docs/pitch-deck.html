03 The insight everyone misses

The amounts are never spoken.
They're on the screen.

Transcript-only tools guess quantities and get them wrong. RecipeReel fuses three signals — and never invents a number.

SPEECH →

The steps

Narration gives the actions, technique, and ordering. Whisper transcribes it with timestamps.

ON-SCREEN →

The quantities

The authoritative amounts live in overlay cards. A vision model reads them verbatim.

TIME →

The deep links

Every step is pinned to the exact second, so a tap jumps you into the video.

Unknown amount? quantity = null, not a hallucination.

04 How it works

Heavy perception on the GPU.
Structured reasoning with Gemma.

Ingest

yt-dlp + ffmpeg

Fetch video & metadata; extract audio and scene keyframes with exact timestamps.

Listen

Whisper-large-v3

Speech → timestamped transcript.AMD MI300X · ROCm

See

Qwen2.5-VL

Frames → captions + on-screen text (the quantities).AMD MI300X · ROCm

Reason

Gemma 4

Fuse everything → one schema-constrained Recipe (temp 0). json_schema

Serve

FastAPI

Validate, persist, stream progress. REST + live SSE.

Every stage is a pluggable provider with CPU / mock fallback — the pipeline degrades, it never crashes.

05 Use of AMD platforms

Real work on real silicon.

The MI300X's 192 GB holds Whisper-large-v3 and Qwen2.5-VL resident at once — the throughput-bound perception that justifies the hardware. Gemma does the reasoning, hosted on AMD via vLLM to compete for Best AMD-Hosted Gemma.

192GBHBM3, both models resident
2GPU workloads: ASR + VLM
1codebase, CPU→MI300X

ROCm masquerades as CUDA

The same device="cuda" code runs on a laptop, then unchanged on the MI300X. Perception on-GPU; Gemma on Fireworks or self-hosted on AMD. Nothing is faked.

06 It's built, not a mockup

A production-shaped backend
that runs in one command.

Async job engine

Queue → worker → live SSE progress. SQLite persistence for jobs and recipes.

Clean REST API

Submit URL/upload, poll, stream, fetch recipe, export schema.org JSON-LD. Swagger at /docs.

Containerized

Slim Dockerfile + an MI300X ROCm image. MIT-licensed, public repo, README.

Pluggable providers

Whisper / Qwen-VL / Gemma swap by env var. Missing key or GPU → graceful fallback.

Offline demo mode

MOCK_MODE=true runs the whole pipeline with zero keys — the demo never hard-fails.

Tested & linted

Pytest suite green, ruff-clean, verified end-to-end in Docker.
